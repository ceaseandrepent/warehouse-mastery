warehouse-mastery

Расширение собирает в таблицу информацию о количестве ресурсов в городах игрока.
Установка:

  1. Копируем папку с расширением (та самая, в которой лежат файлы README.html, manifest.json и др.) из архива в любую другую удобную папку. например: d:\chrome-extensions\warehouse-mastery.

  2. В хроме направляем путь свой в меню расширений.

  3. Ставим галочку Режим разработчика.

  4. Нажимаем Загрузить распакованное расширение... и там выбираем папку из п.1 (в этой папке лежит manifest.json и др.).

  5. После обновления страницы икариама в адресной строке появится значок расширения.
